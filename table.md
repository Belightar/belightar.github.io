
# 数据库数据字典

## 概述

本文档是数据库数据字典，用于描述数据库中的表结构，字段信息，以及字段之间的关系。

## 目录

- [用户表](#user_table)
- [订单表](#order_table)
- [商品表](#product_table)

## 用户表

| 字段名称 | 数据类型 | 主键 | 描述 |
| :--- | :--- | :--- | :--- |
| id | int | 是 | 用户id |
| name | varchar(20) | 否 | 用户名称 |
